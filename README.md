# Cyberpunk Portfolio

Personal portfolio website with cyberpunk terminal aesthetic.

🚀 Built with React, TypeScript, and Tailwind CSS
✨ Featuring particle effects, animations, and modern UI components

## Features
- Hero section with typing animation
- Animated statistics counters
- Project showcase
- Skills matrix with progress bars
- YouTube videos and blog posts
- Contact section

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

The built site will be in `dist/public` directory.

## Analytics

The production build supports Google Analytics 4. Set the GitHub Actions repository variable `GA_MEASUREMENT_ID` to your GA4 measurement ID, for example `G-XXXXXXXXXX`.
